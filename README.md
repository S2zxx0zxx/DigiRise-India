## ⚡ Project Overview

> **DigiRise India** ek full-fledged **Digital Marketing Agency Website** hai — jo Indian small businesses ke liye build ki gayi hai. Ek single HTML file mein poora ek-page website — zero dependencies, zero frameworks, blazing fast.

```
🎯 Target Audience  →  Indian Small Business Owners (Kirana to Clinics)
🛠️ Tech Stack       →  Pure HTML5 + CSS3 + Vanilla JS
📦 Size             →  Single .html file (~1.5 MB with embedded images)
🌐 Deployment       →  GitHub Pages / Any Static Host
📱 Responsive       →  Mobile-first, works on all screen sizes
```

```bash

│
├── 📄 index.html                  # Main website (single-page)
```

## ⚙️ How to Run

```bash
# Option 1 — Direct open (fastest)
Just open agency-website-v6.html in any browser. Done. ✅

# Option 2 — GitHub Pages deploy
1. Fork this repo
2. Go to Settings → Pages
3. Source: main branch / root
4. Your site is live at: https://yourusername.github.io/digirise-india/

# Option 3 — Local server
npx serve .
# or
python3 -m http.server 8080
```
