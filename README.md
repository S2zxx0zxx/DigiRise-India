## ⚡ Project Overview

> **DigiRise India** ek full-fledged **Digital Marketing Agency Website** hai — Kolkata-based agency jo Indian small businesses ke liye build ki gayi hai. Ek single HTML file mein poora ek-page website — zero dependencies, zero frameworks, blazing fast, aur deep SEO/AI-crawler optimization ke saath.

```
🎯 Target Audience  →  Indian Small Business Owners (Kirana to Clinics to D2C brands)
🛠️ Tech Stack       →  Pure HTML5 + CSS3 + Vanilla JS
🌐 Deployment       →  Vercel (digi-rise-india.vercel.app)
📱 Responsive       →  Mobile-first, real iPhone-style mockups, dark mode supported
```

```bash

│
├── 📄 index.html                  # Main website (single-page, v7)
```
